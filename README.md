# dotfiles

## Setup

Step 1: Install Oh My Zsh

```sh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

Step 2: Clone repo

```sh
git clone https://github.com/chrisx8/dotfiles.git
```

Step 3: Run `./setup.sh`
